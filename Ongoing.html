Application of Data Science techniques to Geophysical Workflows

Through the knowledge obtained from Data Science studies, we have been able to revisit familiar workflows and apply techniques for quick QC and interactive displays. Tools used for this include Python, Pandas, Numpy, and other associated visualization libraries like Matplotlib etc, Seaborn. Integration of AI and Deep Learning to Geophysical Workflows

With Artificial Intelligence and Deep Learning impacting geophysical workflows, analysis of several workflows associated with Neural Nets, Deep Learning were examined. Tools like Keras, Tensorflow PyTorch have been applied using Open Source data. Data sources include Kaggle, FORCE,
